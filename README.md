# 🎬 VibeCast – Free Next.js Streaming Template

A beautiful and modern **streaming UI template** built with **Next.js 15**, **Tailwind CSS**, and **ShadCN UI**.

Perfect as a starting point for building:
- 🎥 Video platforms
- 📺 Content portals
- 📈 SaaS MVPs or demos
